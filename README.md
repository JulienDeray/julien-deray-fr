# julien.deray.fr

Personal site — plain HTML, CSS, JS. No frameworks, no build step.

## Deploy

Push to `main`. GitHub Pages serves from the root. Custom domain: `julien.deray.fr`.

## Structure

```
├── index.html       # Single-page site
├── style.css        # All styles
├── assets/          # Images, fonts
├── CNAME            # Custom domain
└── README.md
```

## Edit

Edit `index.html` directly. The Leader's Tools section is designed to be a living list — add books, methods, and frameworks as you discover them.
